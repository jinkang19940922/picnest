## 📌 测试执行备注

### 前置条件
- Docker Compose 服务已启动（`docker-compose up -d --build`）
- 容器运行正常：`docker ps` 显示 picnest-backend、picnest-frontend、picnest-nginx
- 测试前需确认服务端口 8080 可访问

### 通过标准
- 高优先级用例（SMOKE-*/AUTH-001/002/UPLOAD-001/002/003/GALLERY-001/002/TRASH-001/002/SETTINGS-001）**必须全部通过**
- 中优先级用例失败需评估影响，低优先级用例允许失败

### 工具建议
- 浏览器测试：Chrome DevTools / Playwright / Cypress
- API 测试：curl / Postman / HTTPie
- 图片素材：准备 3-5 张测试用 JPG/PNG 图片（< 10MB）
- 远程图片 URL：可使用 https://httpbin.org/image/jpeg
